# design-system-baixa-visao

Repositório do sistema de design para baixa visão, com foco em acessibilidade, reutilização e rastreabilidade entre diagnóstico, diretrizes e componentes.

## Problema
Pessoas com baixa visão enfrentam barreiras recorrentes em contraste, legibilidade, navegação e previsibilidade de interface. Este projeto organiza decisões de design para reduzir essas barreiras com base em critérios de acessibilidade (WCAG).

## Como reutilizar
1. Use os arquivos de `tokens/` como base de tema e tipografia.
2. Consulte `docs/` para regras, estados e mapeamento com WCAG.
3. Reaproveite o protótipo e diagramas para guiar implementação.
4. Adapte a pasta `pesquisa/` para registrar validações e evidências no seu contexto.

## Estrutura do repositório
- `docs/`: guia de estilo navegável.
- `tokens/`: design tokens em JSON e CSS.
- `prototipo/`: referências de protótipo (Figma e exportações).
- `diagramas/`: mapa do site e fluxos.
- `pesquisa/`: artefatos de diagnóstico e avaliação.
- `artigo/`: documento final do TCC.
