# Diagramas

Diagramas que estruturam a arquitetura da informação e os fluxos de tarefa do protótipo.

## Arquivos

### Mapa do site
`mapa-do-site.png`

Evidencia a arquitetura da informação e a relação hierárquica entre as áreas funcionais: Página Inicial, Conteúdo/Serviços, Contato, Busca, Formulários e Menu de Acessibilidade.

### Fluxogramas de tarefa

Cada fluxograma detalha caminhos de interação, decisões do usuário e critérios WCAG verificados em cada etapa.

- `fluxo-busca-e-localizacao.png` — Busca e localização de conteúdo
- `fluxo-preenchimento-formulario.png` — Preenchimento de formulário acessível
- `fluxo-ativacao-recursos-acessibilidade.png` — Ativação e ajuste de recursos de acessibilidade

## Observação

Os fluxos foram selecionados por representarem situações críticas para a baixa visão: localizar informação por meio de busca, preencher formulários e ajustar preferências visuais. Para garantir legibilidade, cada fluxo é apresentado em arquivo próprio.
